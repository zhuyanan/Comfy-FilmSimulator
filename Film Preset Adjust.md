这是一个胶片预设（JSON）参数的完整速查表，按功能分类，方便调整和查阅。

### 1. 基础与物理质感 (Base & Physics)

| 参数名 | 数据结构 | 含义与作用 | 调整方向 (数值变大 ↑) |
| :--- | :--- | :--- | :--- |
| **type** | 字符串 | 胶片类型 | `"color"` (彩色) 或 `"bw"` (黑白) |
| **sens_factor** | 浮点数 | **感光度因子**<br>模拟胶片的 ISO 敏感度。 | **↑**：光晕(Halation)更强，颗粒在中间调更明显。<br>**↓**：画面更干净、锐利。 |
| **grain_base** | 数组 [4] | **颗粒结构定义**<br>`[红, 绿, 蓝, 串扰]` | **[0-2]**：R/G/B 各通道的颗粒强度。<br>**[3] 串扰 (Crosstalk)**：不同颜色颗粒的粘连程度。值越大，颗粒越厚实、更有机。 |
| **saturation** | 浮点数 | **基础饱和度** (可选)<br>胶片的化学色彩浓度。 | **↑**：色彩更浓郁 (如 Velvia)。<br>**↓**：色彩更平淡 (如电影负片)。默认 1.0。 |

---

### 2. 光学响应 (Optical Response)
定义红、绿、蓝三层乳剂各自的物理特性。
*彩色用 `opt_r`, `opt_g`, `opt_b`；黑白用 `opt_l`*。

| 数组位置 | 参数名 | 含义 | 调整效果 |
| :--- | :--- | :--- | :--- |
| **Index 0** | **Diffusion** | **光晕/散射强度** | **↑**：该颜色的物体发光/柔光更强 (如红色选 1.2 会让肤色发光)。 |
| **Index 1** | **Gain** | **感光速度 (白平衡)** | **↑**：该颜色变亮/变多 (偏色)。<br>例如红 Gain 高，画面偏暖。 |
| **Index 2** | **Gamma** | **反差/对比度** | **↑**：该颜色的对比度增加，色彩更纯、更深沉。<br>**↓**：该颜色变灰、柔和。 |

---

### 3. 色彩校准矩阵 (Matrix)
定义胶片如何“看见”颜色（光谱灵敏度）。

| 数组范围 | 类型 | 含义 | 调整技巧 |
| :--- | :--- | :--- | :--- |
| **0 ~ 8** | **彩色** | **RGB 混合权重**<br>前9个数字。 | **对角线 (0,4,8)**：大于 1.0 增加饱和度。<br>**非对角线**：控制偏色 (如绿色里混入蓝色)。 |
| **9 ~ 11** | **黑白** | **滤镜权重**<br>后3个数字。 | **[9]红**：调大则肤色白、蓝天黑。<br>**[10]绿**：调大则树叶白。<br>**[11]蓝**：调大则天空白、肤色黑。 |

---

### 4. 影调曲线 (Curve - Hable)
控制画面的明暗对比和动态范围。

| 参数 | 全称 | 作用区域 | 调整口诀 |
| :--- | :--- | :--- | :--- |
| **A** | Shoulder | **高光滚降** | **↑**：高光变柔 (不刺眼，有层次)。<br>**↓**：高光变硬 (数码味)。 |
| **B** | Linear Strength | **整体亮度** | **↑**：画面变亮、通透。<br>**↓**：画面变暗、浓郁。 |
| **C** | Linear Angle | **中调硬度** | **↑**：对比强，脆。<br>**↓**：对比弱，软 (Log感)。 |
| **D** | Toe Strength | **暗部范围** | **↑**：暗部发灰、软绵绵。<br>**↓**：暗部扎实。 |
| **E** | Toe Numerator | **暗部细节** | **↑**：提亮死黑区，找回细节。<br>**↓**：纯黑。 |
| **F** | Toe Denominator | **黑位压缩** | **↑**：强力压暗阴影 (Crush Blacks)。 |
| **gamma**| Gamma | **全局反差** | **↑**：画面深沉，对比大。<br>**↓**：画面灰亮，低对比。 |
| **exposure_bias** | Exposure | **曝光补偿** | **↑**：整体提亮 (EV)。 |

---

### 5. 艺术调色 (Tint / Split Toning)
控制高光和暗部的染色。

| 参数名 | 数据结构 | 含义 | 调整效果 |
| :--- | :--- | :--- | :--- |
| **shadows** | [R, G, B] | **暗部染色** | **正数**：提升黑点并染色 (如 `[0,0,0.05]` 暗部偏蓝)。<br>让黑色不黑 (Fade Look)。 |
| **highlights**| [R, G, B] | **高光染色** | **负数**：压暗白点并染色 (如 `[-0.05,-0.05,-0.1]` 高光偏黄)。<br>让白色不刺眼 (Matte White)。 |
| **range** | 浮点数 | **响应范围** | **↑ (3.0)**：只染极黑/极白，肤色干净。<br>**↓ (1.0)**：染色覆盖全图，像滤镜。 |
